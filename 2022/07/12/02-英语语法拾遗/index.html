英语语法拾遗

发表于 2022-07-12 更新于 2022-07-31 分类于 skills

写在前面

​ 恶补听力和口语过程中发现，造句能力弱化，决定快速过一遍简化的英语语法，目标是够用就好，以下是关键点记录。

​ 注意：忽略了很多的特例。

英语语法梗概

句子成分

词性

谓语动词（句子成分）

英语 谓语动词的特性是中文所没有的

时态 由 时间和状态 排列组合而成。

语气也很重要

助动词

帮助谓语动词完成意思的表达，如否定，完成等

​ 助动词通常也可以用作其他含义，需要区分。

非谓语动词

非谓语动词，可以看做是谓语动词改动的，可以充当除谓语动词外的所有句子成分，取代几乎所有从句简化句子。但是非谓语动词丧失了谓语动词的 时间，状态和预期等本领，而是以其他形式存在。

非谓语动词存在很多形式，也极易混淆。

动词分类

简单句有五大句型。

简单句有五大句型，这也区分了五大类谓语动词。

一个简单句只能有一个谓语动词，带有时体气等性质；其他的动词只能以非谓语动词的形式出现，无法表达时体气。

由是否需要宾语补语可以划分为完全及物动词和不完全及物动词。

语法体系很多，词性归类够用就好。

动词时态

​ 国内传统分类体系，即4*4

动词时间

过去，现在，将来 和 过去将来。

过去将来和现在没有必然关系，画图如下

但是也可能这样

动词状态

一般，进行，完成，完成进行

最简单的进行，表示动作正在发生。

完成态注重动作已经完成，并没有指出具体的开始时间。

动作的一部分完成了，但是剩下的在进行中。

一般的，默认的，简单的，没有具体说明状态信息，就是一般状态

时态难点主要有四个：时与态分开，动词变位，助动词（也要动词变位）和 其他语法混合（情态动词+时态）

常用时态

一般现在时

+动词原型

+动词原型 + s (第三人称单数)

表示 事实，习惯/重复动作，预计发生的事情（时刻表等）

错误：I eat a carrot. 一般现在时缺少时间信息，没有一个胡萝卜可以一直吃，只能说一般状态，

现在进行时

+助动词be的变位+ 动词的现在分词

​ I am; you/we are; he/she/it is

​ be 可以作为连系动词（被动态），助动词

​ 现在分词：通常+ing

现在完成时

现在已经完成的动作，往往对现在产生影响

+助动词have的变位+动词的过去分词

​ I/you/we have ; he/sh e/it has

​ 过去分词：通常+ed

注意：过去式和过去不一样

现在完成进行时

+have的变位+been+动词的现在分词

​ I/you/we have ; he/sh e/it has

一般过去时

+动词过去式

动作发生了，但是不知道其他信息

过去的习惯动作： used to

过去进行时

+助动词be的变位+ 动词的现在分词

I/he/she/it was; you/we were

注意点：

第一张是图片，第二张是动图

过去完成时

+助动词have的变位+动词的过去分词

I/you/we/he/she/it had

过去A的过去B发生了事情，对过去A有影响。

过去完成进行时

+助动词have的变位+been+动词的过去分词

I/you/we/he/she/it had

一般将来时

+will + 动词原形

将来进行时

+will+be+动词的现在分词

在将来某个时间，某个动作在进行状态中。

将来完成时

+wiil+have+动词过去分词

对于将来某个时间点，已经完成，且对将来有影响

将来完成进行时

+will+have+been+动词的现在分词

对于将来某个时间点，已经完成但是还要继续的动作

一般过去将来时

与现在无关，出现在过去时态的从句中

+ would + 动词原形

would 是 will 的过去式

或者 was/were + going to do

注意：would 多义，可以构成虚拟语气，表示礼貌

过去将来进行时

+ would + be + 动词的现在分词

过去将来完成时

+ would + have + 动词的过去分词

过去将来完成进行时

+ would + have + been + 动词的现在分词

动词的语气

陈述语气

​ 描述现实

疑问语气

​ 问句，改变词序，加助动词

祈使语气

 命令/请求

​ 第二人称的一般现在时，并去掉you。

虚拟语气

​ subjuntive mood，与现实相悖的，主观意愿，多用于从句中

  • 做不可能的假设

    与现在相反：条件句：过去式，主句：would+动词原形

​ 与将来相反: 条件句：should/were to +动词原形，主句：would+动词原形。

​ would可改为 should/could/might

​ 与过去相反：条件句：had+动词过去分词，主句：would+have+动词过去分词

  • 表达愿望，请求，建议，命令，希望某个动作/时间发生

    wish 可能性小的事发生（hope 中性），would rather，

    对现在的愿望：wish + 动词过去式

    对过去的愿望：wish + had / （would have） +动词过去分词

    对将来的愿望：wish + would/ could/ might/ should + 动词原形

    demand，suggest，insist，order + should 动词原形 （任何时态）

从句

一个句子作为另一个句子的句子成分。（主从关系）

从词性分类

形容词从句

关系词可以分为关系代词和关系副词，关系副词 = 介词+关系代词

限定性形容词从句 和 非限定性形容词从句，逗号区分，插入语。非限定性形容词从句可以指代整句话

名词性从句

主语从句
